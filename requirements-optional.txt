# Optional dependencies for Nucleo
# Install these if you need specific features
# WARNING: Some of these may have dependency conflicts with core requirements.txt
# Install in a separate virtual environment if needed

# Web search tool (has dependency conflict with httpx - see notes below)
# brave-search==0.1.8 is the last stable version
# Note: brave-search requires httpx<0.26.0, but core requires httpx>=0.27.0
# To use: pip install brave-search==0.1.8 --force-reinstall httpx==0.25.2
# Or use in a separate virtual environment

# If you need web search, you have two options:

# Option 1: Create a separate venv for search functionality
# python -m venv venv-search
# source venv-search/bin/activate  # or venv-search\Scripts\activate on Windows
# pip install brave-search==0.1.8

# Option 2: Downgrade httpx (may affect performance)
# pip install httpx==0.25.2 brave-search==0.1.8
# Note: This will downgrade httpx from 0.27.0 to 0.25.2
